# GTNH Power Monitor

Tiered power-grid dashboard cover for GT New Horizons 2.9.0-beta-2.

## Requirements

**Requires GTNH 2.9.0-beta-2 or later.** Earlier packs — including 2.9.0-beta-1 and all 2.8.x versions — will crash at startup. The mod is built against GT5-Unofficial 5.09.54.20, ModularUI2 2.3.79, and GuideNH 1.3.11, which first ship together in beta-2.
